## Development

Run the development server.

```bash
npm run dev
```

## Build

Build the project for production.
```bash
npm run build
```

## Start

Start the production server.

```bash
npm run start
```

## Lint

Lint the TypeScript react files using ESLint

```bash
npm run lint
```
## Format

Format files using Prettier.

```bash
npm run format
```


## Test

Run tests using Jest.

```bash
npm run test
```
